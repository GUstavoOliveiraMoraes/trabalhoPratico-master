Getting Started

Welcome

Obrigado por adquirir o sistema LocaPlus.

Licença

Características

Conheça as ferramentas do sistema:

  • Gerenciamento de Automóveis
  • Gerenciamento de Clientes
  • Gerenciamento de Locação de Automóveis
  • Gerenciamento de Funcionários
  • Gerenciamento de Contas da empresa(inovação)

Requisitos

Requisitos do sistema:

  • PHP 7 ou superior
  • MYSQL ou MariaDB
  • Apache ou NGINX

Instalação

Para efetuar a instalação do LocaPlus, segue esses passos:

  • Extraia a pasta LocaPlus;
  • Envia todos os arquivos da pasta Upload para a pasta public_html ou www;
  • Cria um banco de dados e dê todas as permissões para o mesmo;
  • Acessa o phpMyAdmin e importa o arquivo db.sql que está dentro da pasta Banco de Dados;
  • Informe os dados do banco de dados nos arquivos: config.php e conexao.php;
  • Está pronto!
